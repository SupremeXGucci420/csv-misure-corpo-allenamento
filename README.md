# csv-misure-corpo-allenamento
Programma python per tenere traccia delle proprie misure corporee e peso durante i propri allenamenti

Campi:
      -Data pesata
      -Peso
      -Media(non usato)
      -Data misure
      -Vita
      -Torace
      -Braccio sx
      -Braccio dx
      -Coscia sx
      -Coscia dx
      -Culo
